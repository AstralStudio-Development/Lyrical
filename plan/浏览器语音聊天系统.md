# Lyrical - 浏览器语音聊天系统实现计划

## 项目概述

创建一个浏览器语音聊天系统，让 Minecraft 玩家无需安装模组，通过浏览器即可进行语音通话。
采用类似 Simple Voice Chat 的服务器中继架构，使用 WebSocket 传输音频数据。

## 技术栈

| 组件 | 技术 |
|------|------|
| MC 插件 | Paper API 1.21.4 |
| WebSocket | Netty (内嵌) |
| Web 前端 | 原生 JS |
| 音频编码 | WebCodecs Opus (支持时) / PCM (回退) |
| 音频处理 | Web Audio API |
| HTTP 服务 | Netty (内嵌) |

## 核心功能

### 1. 近距离语音
- 玩家之间的声音随距离衰减
- 3D 空间音频 (HRTF)

### 2. 群组功能
- 创建/加入/离开群组
- 群组内语音不受距离限制
- 群组密码保护

### 3. 玩家状态
- 静音/取消静音
- 禁用/启用麦克风
